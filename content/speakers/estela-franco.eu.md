{{<speaker_bio>}}

Shopify-n web errendimenduan aditua den senior teknologia-arkitektoa, eskala handiko proiektuetan abiadura eta erabiltzaile-esperientzia optimizatzean espezializatua. Web Errendimendua, Web Estandarrak eta TechSEO gustuko ditut; Google Developer Expert naiz Web Teknologietan, nazioarteko hizlaria, #MujeresEnSEO-ren sortzailekidea eta PerformanceObserver.dev-eko antolatzailekidea, web komunitate globalari aktiboki laguntzen.

{{</speaker_bio>}}

{{<speaker_talk title="Web Errendimenduaren Hondamendi Handienak">}}
Igo errendimenduaren bolumena! Saio honetan, atzo eta gaurko arrakasta handiak berrikusi egingo ditugu, zure proiektuaren karga-abiadura, erabiltzaile-esperientzia eta negozio-metrikak suntsitzen ari direnak (eta nola saihestu!). Beste hainbat hit artean, JavaScripten melodia hiltzaileak, karga asinkronoaren balada geldoak eta hari nagusia blokeatzen duten riffak identifikatzen ikasiko duzu. Aurkitu nola utzi porrotaren abestia jotzeari eta zure proiektua errendimenduaren headliner bihurtu.
